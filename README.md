# Nightbar

Nightbar is a cool, retro light bar effect written in JavaScript

## Example

    <script src="/nightbar.js"></script>
    <div id="nightbar"></div>
    <script>
      var nightbar = document.getElementById('nightbar');
      nightbar.appendChild(Nightbar());
    </script>
